# 📊 Dashboards de Inteligencia de Negocios (BI) para Retail

**Descripción**
Repositorio que recopila tableros interactivos y reportes administrativos creados para la toma de decisiones estratégicas, el análisis de gastos y el seguimiento de indicadores clave de rendimiento (KPIs) en tiendas de retail.

**✨ Áreas de Análisis**
* **Dashboard de Ventas y Márgenes:** Visualización de la rentabilidad por departamento y comportamiento de la facturación electrónica.
* **Control de Gastos y Operaciones:** Monitoreo de costos operativos, eficiencia logística en rutas de reparto y mermas.
* **Seguimiento de Metas:** Gráficos dinámicos para evaluar el cumplimiento de objetivos del personal y de la tienda.

**🛠️ Herramientas Utilizadas**
* Power BI
* Microsoft Excel Avanzado (Tablas dinámicas, Macros, Fórmulas lógicas)

**🚀 Objetivo del Proyecto**
Transformar datos operativos de puntos de venta (POS) y facturación en información visual clara, permitiendo a la gerencia identificar oportunidades de mejora continua y control de procesos.
